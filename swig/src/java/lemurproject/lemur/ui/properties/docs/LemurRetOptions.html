Lemur Retrieval GUI - Options

[« Back to Main]

The base options screen (shown below) has several options. In the general settings, you can choose to have any currently opened indexes automatically load the next time the program starts. Also, by default, when you click on a document to display it, it will open in the same window as any previous documents - you can choose to display each document in its own window by check the box entitled "Open each document in its own window"

Futhermore, you can choose to highlight the matched query terms and display HTML markup tags within the search results and documents window as well. Finally, in the general options, you can also choose to append to existing files when exporting results, rather than the default of overwriting the file.

In the "Retrieval" options, you can set the number of results per page, as well as the retrieval method to use while searching.

There are four different retrieval models to choose from:

  • KL-Divergence (using Dirichelp Prior smoothing)
  • Okapi (with settable parameters for BM25 settings and query expansion)
  • InQuery (for structured queries, with settable options)
  • Indri Query Language (for Indri Style Queries)

Note that only the Indri-Style indexes can be used with the Indri Query Language option above
